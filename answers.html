The internet backbone is the core of the internet in which there are millions of interconnected computer networks. These data routes are hosted by commercial, academic, and government as well as other high capacity networking centers A client is like a customer that makes a request and then the server answers the request by fulfilling it Servers store and distribute data, routers direct traffic between networks, switches connect devices within a network, and workstations are user computers. Together, along with other hardware components, they form the infrastructure of the internet, enabling global communication and data exchange. The Internet Protocol Suite, also known as TCP/IP, is a framework of protocols that enables communication between devices over the internet. It consists of multiple layers, each responsible for different aspects of data transmission.

  • Web 1.0: This era, from the 1990s to the early 2000s, was characterized by static web pages providing one-way communication, mainly for consuming information.
  • Web 2.0: Emerging in the mid-2000s and continuing to the present, Web 2.0 introduced dynamic, interactive platforms allowing user-generated content, social interaction, and collaboration
  • Web 3.0: Currently in its early stages, Web 3.0 aims to create a more intelligent, interconnected, and decentralized web ecosystem, leveraging technologies such as semantic web, artificial intelligence, and blockchain
Web 1.0 was static and one-way, Web 2.0 introduced dynamic and interactive user-generated content, and Web 3.0 is evolving towards a more intelligent, interconnected, and decentralized web ecosystem leveraging technologies like AI and blockchain. Each stage represents a significant shift in the way people interact with and use the internet. Markup languages play a crucial role in the Web by providing a standardized way to structure and format content. Some key aspects can be seen as structuring content, formatting and presentation, accessibility , interactivity, and cross platform compatibilaty Markup languages structure and format content on web pages, using tags to define elements. They are static and declarative. Examples include HTML and XML. Scripting languages add dynamic functionality and interactivity to web pages using code. They are dynamic and procedural, executed by browsers or servers. Examples include JavaScript and PHP. SGML laid the groundwork for HTML and XML. HTML is used for structuring web content, while XML is used for data storage and transport. XHTML combines HTML syntax with XML rules, making it stricter and more compatible with XML. DHTML refers to a combination of HTML, CSS, and JavaScript for creating dynamic web pages, while Markdown offers a simpler syntax for formatting text-based documents.
